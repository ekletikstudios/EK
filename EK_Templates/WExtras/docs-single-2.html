ReactJS: Vale a Pena?

Escrito por Paulo J.

React is all about rendering out some sort of element, say an h1 that says "Hello," and then targeting something on the DOM, like the document body. You can see once we've done that and I hit refresh, that we have "Hello" already rendered out. 

                
                  import React from 'react';

                  class App extends React.Component {
                  render(){
                  return <\h1>Hello World!<\/h1>
                  }
                  }//end App class


                  // Alternative way of doing the same, but with a stateless function
                  // const App = () => <\h1>Hello Stateless World!<\/h1>

                  export default App

To create an actual component, I'll delete this line. I'll say "div" with an h2 inside of it...
